# MoneyDevKit Expert Services

Quick deploy to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/mdk-services)

## Local Development

```bash
cd landing-page
npx serve .
```

## Files

- `index.html` - Main landing page (self-contained, no build needed)
- `package.json` & `next.config.js` - For Vercel deployment

## Customization

Edit `index.html`:
- Change contact email (search for "your-email@example.com")
- Update pricing
- Modify services
- Add your Calendly link
